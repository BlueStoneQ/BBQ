# 算法速查手册（🔥优先题）

> 每题一句话核心解法，复习时对着 algo 项目看代码。

## 双指针

| 题 | 核心解法 |
|----|---------|
| 26 删除有序数组重复项 | 快慢指针，不同则 fast 覆盖 slow+1 |
| 27 移除元素 | 快慢指针，不等于 val 则 fast 覆盖 slow |
| 88 合并两个有序数组 | 从后往前双指针填充，避免覆盖 |
| 15 三数之和 | 排序 + 固定一个 + 左右夹逼，跳过重复 |
| 21 合并两个有序链表 | 虚拟头 + 比较取小的接上 |

## 滑动窗口

| 题 | 核心解法 |
|----|---------|
| 209 长度最小的子数组 | right 扩窗口直到 sum≥target，left 缩窗口更新最小长度 |
| 3 无重复最长子串 | Set 记录窗口内字符，重复则 left 缩到不重复 |

## 二维数组

| 题 | 核心解法 |
|----|---------|
| 59 螺旋矩阵 II | 四边界逼近，每圈缩一层 |
| 240 搜索二维矩阵 II | 从右上角开始，大了左移，小了下移 |

## 链表

| 题 | 核心解法 |
|----|---------|
| 206 反转链表 | pre/cur/next 三指针，cur.next = pre 逐个翻 |
| 92 反转链表 II | 找到 left 前驱，局部反转，接回去 |
| 21 合并有序链表 | 虚拟头 + 比较取小 |
| 141 环形链表 | 快慢指针，相遇则有环 |
| 142 环形链表 II | 相遇后一个回 head，同速走再相遇就是入口 |

## 栈

| 题 | 核心解法 |
|----|---------|
| 20 有效的括号 | 遇左括号压栈，遇右括号弹栈比较 |

| 215 第 K 大元素 | 快速选择（partition 到 k 位置）或小顶堆 |

## 树

| 题 | 核心解法 |
|----|---------|
| 102 层序遍历 | BFS 队列，每层 size 控制 |
| 105 前序+中序构造树 | 前序第一个是根，中序分左右子树，递归 |
| 106 中序+后序构造树 | 后序最后一个是根，中序分左右，递归 |
| 236 最近公共祖先 | 后序递归，左右都找到则当前是 LCA |
| 235 BST 最近公共祖先 | 都小走左，都大走右，分叉点就是 |
| 297 序列化/反序列化 | 前序遍历序列化，用 # 标记 null，递归反序列化 |
| 144 前序遍历迭代 | 栈：弹出处理，先压右再压左 |

## DP 动态规划

| 题 | 核心解法 |
|----|---------|
| 509 斐波那契 | dp[i] = dp[i-1] + dp[i-2] |
| 70 爬楼梯 | 同斐波那契 |
| 746 最小花费爬楼梯 | dp[i] = min(dp[i-1]+cost[i-1], dp[i-2]+cost[i-2]) |
| 62 不同路径 | dp[i][j] = dp[i-1][j] + dp[i][j-1] |
| 63 不同路径 II | 同上，障碍格 dp=0 |
| 343 整数拆分 | dp[i] = max(j*(i-j), j*dp[i-j]) |
| 53 最大子数组和 | dp[i] = max(dp[i-1]+nums[i], nums[i])，连续子数组 |

## 模式速查

| 模式 | 适用场景 | 关键词 |
|------|---------|--------|
| **双指针** | 有序数组去重/合并/三数之和 | 快慢指针 / 左右夹逼 |
| **滑动窗口** | 最长/最短子串/子数组 | right 扩 left 缩 + 条件判断 |
| **二分** | 有序区间求最值/边界 | left/right/mid + 缩半 |
| **BFS** | 最短路径/层序遍历 | 队列 + visited |
| **DFS/回溯** | 排列/组合/子集/分割 | 递归 + path + 撤销选择 |
| **DP** | 最优子结构 + 重叠子问题 | 状态定义 → 转移方程 → 初始化 → 遍历顺序 |
| **贪心** | 局部最优 = 全局最优 | 排序 + 每步取最优 |
| **单调栈** | 下一个更大/更小元素 | 栈维护递减/递增序列 |
| **前缀和** | 频繁查询区间和 | preSum[i] = sum(0..i) |
