# JY - 前端开发（AI大模型方向）

## 岗位职责

1. 负责 Web、跨端容器（移动或桌面）中的 **AI 大模型应用开发**，保障代码质量和用户体验。
2. 以不断提升质量、效率、体验为目标，封装组件、沉淀文档、生产工具、搭建系统，践行工程师日常工作准则。

## 任职要求

1. 2年以上前端开发经验。
2. 掌握 Web 标准和核心技术（Javascript、HTML、CSS），熟悉 Flex 等页面布局方案，理解表现与数据分离的开发思想。
3. 熟练掌握至少一种 MVVM 视图框架（React、Angular、Vue 等）。
4. 熟悉 HTTP 协议，能熟练使用相关网络调试工具开展问题排查。

## 加分项

1. 具备实际**跨端技术融合**实践经验，包含但不限于 RN、Hybrid Web、Electron 等技术栈。
2. 了解 GPT 等大模型技术原理，对大模型工程化应用、**LangChain 等 Agent 方案**有深入理解及实践经验。

| 维度 | 关键点 | 准备方向 |
|------|--------|----------|
| **核心业务** | AI 大模型应用 + Web/跨端容器 | 准备 LLM 应用开发经验（对话、RAG、Agent 等） |

## Prep Checklist

### Must-have (hard requirements)
- [ ] JS 基础：闭包、原型链、事件循环、Promise
- [ ] CSS 布局：Flex 各属性、响应式方案
- [ ] 框架原理：虚拟 DOM、Diff 算法、响应式系统、Hooks
- [ ] HTTP：请求方法、状态码、缓存（强缓存/协商缓存）、CORS
- [ ] 工程化：Webpack/Vite 构建、模块化、组件封装思路

### Bonus
- [ ] AI 应用开发：Streaming 输出、Prompt 工程、Function Calling、RAG 流程
- [ ] Agent 方案：LangChain 架构、Tool Use、Memory 管理
- [ ] 跨端：Electron 主进程/渲染进程通信、RN Bridge 原理
- [ ] 工程师文化：组件库沉淀、文档建设、工具链搭建的实际案例
