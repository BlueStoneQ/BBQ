# JD（脱敏）— 阶跃星辰

> AI 大模型公司，做 AI 应用的前端/跨端开发。

---

## 岗位：前端开发工程师（AI 大模型应用）

**职责**

1. 负责 Web、跨端容器（移动或桌面）中的 AI 大模型应用开发，保障代码质量和用户体验
2. 以不断提升质量、效率、体验为目标，封装组件、沉淀文档、生产工具、搭建系统

**要求**

1. 2 年以上前端开发经验
2. 掌握 Web 标准和核心技术（JS/HTML/CSS），熟悉 Flex 布局，理解表现与数据分离
3. 熟练掌握至少一种 MVVM 框架（React/Angular/Vue）
4. 熟悉 HTTP 协议，能用网络调试工具排查问题

**加分项**

1. 跨端技术融合实践（RN / Hybrid Web / Electron）
2. 了解 GPT 等大模型技术原理，对大模型工程化应用、LangChain 等 Agent 方案有深入理解及实践

---

## 拆解

### 岗位本质

**在 AI 大模型公司做"AI 产品的前端"**——把模型能力包装成用户可用的产品界面（Web/App/桌面端）。

类似于：给 ChatGPT/Claude 做前端的那个角色。

### 核心考察

| 维度 | 权重 | 说明 |
|------|------|------|
| 前端基本功 | ⭐⭐⭐⭐ | React/Vue + 工程化 + 组件封装 |
| AI 应用开发经验 | ⭐⭐⭐⭐ | 流式输出/对话 UI/Markdown 渲染/Agent 交互 |
| 跨端能力 | ⭐⭐⭐ | RN/Electron/Hybrid（加分项但他们看重） |

### 匹配度

**极高（90%+）**：

- 前端 10 年 ✅

- AI Agent 实践（Mako 框架 + MCP 全链路）✅
