## 组合话术

> "我的核心能力是三个层次：
> 第一，性能体验优化——我能从启动/渲染/内存/包体多维度系统化优化，有数据验证，有自动化持续监控；
> 第二，跨层通信架构——我能穿透 JS → C++ → Java/Swift 解决底层问题，不是只会 JS 的前端；
> 第三，工程化体系建设——我能建分 Bundle/热更新/CLI/组件体系，让团队持续受益，不是一次性修 bug。"
