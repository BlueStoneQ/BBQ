## 核心叙事（针对这类岗）

```
三层全栈大前端架构师：

  第一层：JS 运行时（Web + RN + 框架）
    → React/Vue、RN 新架构（Fabric/TurboModule/JSI）、框架设计

  第二层：Native 容器层（Android + iOS）
    → WebView 容器定制、JSBridge 协议、Native Module
    → J2V8 / JNI / WKScriptMessageHandler

  第三层：C++ / 引擎层（通信骨干）
    → JSI HostObject、V8 嵌入、NDK
    → JS 和 Native 之间的"胶水层"

  + AI 贯穿全层：
    → Agent 驱动架构 + MCP 工具链 + AI 友好代码库设计

= 稀缺定位：能设计横跨 JS → Native → C++ 的架构
= 大部分前端 Leader 只覆盖第一层，你覆盖三层
```

## 需要准备的 STAR

1. **XRN 多 Bundle** — 从零设计跨端架构，多 Bundle 热更新，共享包体系

## 原文

### 岗位职责

1. 技术长线规划：负责端（移动端/React Native）与 Web 侧的技术长线规划及技术栈演进路线设计，确保前端架构的前瞻性、可扩展性与稳定性。
2. AI 驱动架构与工程化：具备极强的 AI 工具操控能力，能够以 Agent 为核心生产力完成架构搭建与核心基建。熟悉如何构建对 AI 友好的代码库结构，利用 AI 全自动生成、分析和重构复杂工程代码，取代传统的手工编码模式。
3. 核心基建与公共组件研发：主导前端核心底座、公共框架、CI/CD 自动化流水线及工程化基建的搭建，解决跨平台开发中的深层次性能瓶颈与技术难题。
4. 技术规范与指导：制定前端开发规范、架构标准与安全审计机制，指导高级开发工程师，提升整体团队的技术天花板。
5. 跨端容器与 Hybrid 架构能力：深入理解 App 宿主环境与 Web 运行时的底层通信机制；精通高性能 JSBridge 协议设计与通道优化，能够通过构建高内聚、低延迟的跨端通信底座，抹平 Native 与 Web 的体验与开发差异；具备将 App 端内开发模式向 Web 生态全面对齐的架构落地经验，实现"一端编写、多端无缝复用"的高效动态化研发体系。

### 任职要求

1. 工作年限与架构背景：具备 5-8 年以上前端开发及架构设计经验，有中大型项目从零到一的架构落地成果。
2. 多端架构能力：精通 Web 移动端及主流前端框架，具备长线控制技术演化的成功案例。精通 React Native 技术生态，有深度的跨端架构设计、复杂工程重构及多端代码复用经验。熟悉原生 iOS（Swift/Objective-C）或 Android（Kotlin/Java）开发，具备双端融合架构经验者优先。
3. AI 技术实践：对 AI 协作有深度实践，理解如何将大模型（LLM）能力融入前端工具链或自动化工作流中。
