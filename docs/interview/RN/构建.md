## 一、构建产物总览

```
RN Release 构建产物（Android）:
├── android/app/build/outputs/apk/release/

│       │
│       └── META-INF/                 ← 签名信息
```

## 二、JS 产物：Bundle vs HBC

### 两种模式

| 模式 | 产物 | 大小 | 启动 |
|------|------|------|------|
| **非 Hermes** | `index.android.bundle`（纯 JS 文本）| 大 | 慢（需解析）|
| **Hermes** | `index.android.bundle`（HBC 二进制）| 小 | 快（直接执行）|

### 构建流程

```
Metro 打包:
index.js → 解析依赖图 → Babel 转译 → 打包 → JS Bundle

Hermes 编译（可选）:
JS Bundle → hermesc → HBC Bytecode
```

## 三、静态文件：包含什么

### 1. 图片资源（核心）

```
JS 中引用:
const icon = require('./assets/icon.png');

assets/
├── icon.png          ← 1x（mdpi）
├── icon@2x.png       ← 2x（xhdpi）
├── icon@3x.png       ← 3x（xxhdpi）
└── icon@4x.png       ← 4x（xxxhdpi）
```

**Metro 处理**：
```
1. 扫描 require 语句
2. 根据后缀 @2x/@3x 匹配分辨率
3. 复制到对应 drawable 目录
4. 生成资源 ID 映射（bundle 中是数字引用）
```

**产物位置**：
```
Android APK:
├── res/drawable-mdpi/icon.png
├── res/drawable-xhdpi/icon.png
├── res/drawable-xxhdpi/icon.png
└── res/drawable-xxxhdpi/icon.png

iOS:
├── Images.xcassets/
│   └── icon.imageset/
│       ├── icon.png
│       ├── icon@2x.png
│       └── icon@3x.png
```

### 2. 字体文件

```
assets/fonts/
├── Roboto.ttf
├── IconFont.ttf
└── CustomFont.otf
```

**处理方式**：
```bash
# 1. 配置 react-native.config.js
assets: ['./assets/fonts/']

# 2. 执行链接
npx react-native-asset

# 3. 产物
Android: 复制到 assets/fonts/
iOS: 复制到 Resources，写入 Info.plist
```

### 3. 音频/视频

```javascript
// 小文件：内联或打包
const sound = require('./assets/sound.mp3');

// 大文件：放原生资源，避免打包进 bundle
// 或动态下载，放应用沙盒
```

**处理方式**：
- 小文件：随 bundle 打包到 `res/raw/` 或 `assets/`
- 大文件：原生 `res/raw/` 或下载到 `getFilesDir()`

### 4. JSON 配置

```javascript
const config = require('./config.json');
```

**处理方式**：
- Metro 内联到 bundle 中（变成 JS 对象）
- 或放 `assets/` 目录，运行时读取

### 5. 其他文件

| 类型 | 处理方式 | 产物位置 |
|------|---------|---------|
| `.ttf/.otf` | 字体链接 | `assets/fonts/` |
| `.mp3/.wav` | 小文件打包，大文件放原生 | `res/raw/` 或下载 |
| `.mp4` | 通常不放 bundle，太大 | 下载或原生资源 |
| `.sqlite` | 原生 `assets/` 复制到沙盒 | `assets/` → `getDatabasePath()` |
| `.html` | WebView 加载 | `assets/` 或网络 |
| `.lottie` | Lottie 动画 JSON | `assets/` 或内联 |

## 四、静态文件处理机制

### Metro 资源管线

```
JS 代码中的 require('./assets/icon.png')
        ↓
Metro 解析 AST，识别 Asset 依赖
        ↓
根据平台规则处理：
├── Android: 生成 drawable-mdpi/hdpi/xhdpi/xxhdpi/xxxhdpi
├── iOS: 生成 Images.xcassets 或扁平复制
└── Web: 生成带 hash 的文件名，用于缓存
        ↓
资源路径替换为平台标识（数字或路径）
        ↓
Bundle 中：require(123) // 资源 ID
```

### 关键配置

```javascript
// metro.config.js
module.exports = {
  resolver: {
    // 额外支持的资源扩展名
    assetExts: [
      'png', 'jpg', 'jpeg', 'gif', 'webp', 'svg',
      'ttf', 'otf', 'mp3', 'wav', 'mp4', 'json',
      'lottie', 'db', 'sqlite'
    ],
    
    // 排除某些扩展名（不当作资源）
    assetExtsBlacklist: ['txt'],
  },
  
  transformer: {
    // SVG 转换
    babelTransformerPath: require.resolve('react-native-svg-transformer'),
  },
};
```

## 五、完整构建流程

```
[源代码]
    │
    ├── [JS 层]
    │   ├── Metro 打包
    │   │   ├── 解析依赖图（JS + npm 包）
    │   │   ├── Babel 转译（JSX/TS/ES6+ → ES5）
    │   │   ├── 资源处理（图片/字体复制到原生目录）
    │   │   └── 生成 Bundle
    │   │
    │   ├── Hermes 编译（可选）
    │   │   └── Bundle → HBC Bytecode
    │   │
    │   └── 产物：
    │       ├── index.android.bundle（或 .hbc）
    │       ├── drawable-*/*.png（图片）
    │       └── fonts/*.ttf（字体）
    │
    ├── [Android 原生层]
    │   ├── Gradle 构建
    │   │   ├── 合并 AndroidManifest.xml
    │   │   ├── AAPT 编译资源（res/ → R.java）
    │   │   ├── 复制 assets/（bundle + 字体 + 其他）
    │   │   ├── Java/Kotlin 编译 → DEX
    │   │   └── 打包 APK/AAB
    │   │
    │   └── 产物：
    │       ├── lib/*.so（Hermes + RN 引擎）
    │       ├── assets/*.bundle
    │       ├── res/drawable-*/
    │       └── classes.dex
    │
    └── [iOS 原生层]
        ├── Xcode 构建
        │   ├── Compile Sources
        │   ├── "Bundle React Native code"（生成 main.jsbundle）
        │   ├── Copy Bundle Resources（图片、字体）
        │   └── Link + Sign
        │
        └── 产物：
            ├── main.jsbundle
            ├── *.framework（RN 引擎）
            └── App.app
```

## 六、静态文件清单总结

| 类别 | 文件类型 | 处理方式 | 产物位置 |
|------|---------|---------|---------|
| **图片** | `.png/.jpg/.jpeg/.gif/.webp/.svg` | 多分辨率缩放 | `res/drawable-*/` |
| **字体** | `.ttf/.otf` | 链接复制 | `assets/fonts/` |
| **音频** | `.mp3/.wav/.ogg` | 小文件打包，大文件原生 | `res/raw/` 或下载 |
| **视频** | `.mp4/.mov` | 通常不放 bundle | 下载或原生 |
| **JSON** | `.json` | 内联或打包 | bundle 内或 `assets/` |
| **数据库** | `.sqlite/.db` | 复制到沙盒 | `assets/` → `getDatabasePath()` |
| **Lottie** | `.json` | 打包 | `assets/` |
| **HTML** | `.html` | WebView 加载 | `assets/` |

## 七、关键产物对比

| 产物 | 内容 | 大小 |
|------|------|------|
| `index.android.bundle` | JS 代码（文本或 HBC）| 5-20MB |
| `libhermes.so` | Hermes 引擎 | ~2MB |

| `res/drawable-*` | 图片资源 | 视项目 |

## 八、构建优化点

| 优化 | 方法 | 效果 |
|------|------|------|
| **Bundle 拆分** | RAM Bundle / Inline Requires | 减少启动加载 |
| **图片压缩** | TinyPNG / WebP 转换 | 减小 APK |
| **字体子集** | 只打包用到的字符 | 减小字体文件 |
| **Hermes** | 启用 HBC | 减小 bundle，提升启动 |
| **资源去重** | 相同 hash 只保留一份 | 减小体积 |
| **ProGuard/R8** | 代码混淆压缩 | 减小 DEX |
