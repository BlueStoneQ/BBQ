# DD - Frontend Expert/Leader (Operations)

## Responsibilities

1. 带领泛前端团队负责网约车运营方向平台的需求开发，负责架构设计，搭建高可用、可扩展的技术体系
2. 深入理解运营与增长业务，推动业务标准化改造，沉淀可复用的组件、工具、业务类型等研发资产
3. **规划并落地 AI 开发体系**：引入 AI 编程工具，探索基于业务建模的代码生成、自动化测试，实现 AI 与研发流程的深度融合
4. 参与泛前端技术战略规划，推动技术平台化建设；与产品、后端等团队紧密协作

## Requirements

1. 本科及以上，计算机相关。**5年以上**泛前端研发经验，有团队管理经验
2. 熟练掌握 React + TypeScript，熟悉微前端与 Node.js，掌握前端工程化工具链（Webpack/Vite），具备性能优化和稳定性建设经验；**深度使用 AI Coding 工具**，能拆解任务驱动 AI 完成高效人机协同
3. 具备大型复杂内部系统架构设计与落地经验，**大模型应用和 Agent 工程落地经验优先**
4. 具备判断力和思辨能力，识别 AI 技术边界和潜在风险；善于复盘和知识迁移，持续追踪技术前沿
5. 出色的沟通协作能力，较强的目标感和执行力

## Leadership Narrative

> 这个岗位是**技术专家 + Leader 复合型**，不是纯管理岗。管理叙事要体现"技术 TL 带方向"而非"项目经理排活"。

### 叙事框架（四个维度）

| 维度 | 核心表达 | 经验支撑 |
|------|---------|---------|
| **技术方向把控** | 定方案、选架构、做技术决策 | 搭建系统技术选型、跨端方案选型、AI 体系规划 |
| **研发资产沉淀** | 组件/工具/标准化，降本提效有数据 | 搭建系统（CLI + 物料 + CI/CD）、调试效率 1min47s→10s、活动创建 0.5pd→15s |
| **AI 体系落地推动** | 从工具引入到流程融合，不是玩具是生产力 | Mako Agent 框架 + MCP 全链路（需求→设计→编码→验证→Review→Bug） |
| **全链路质量卡控** | 不是口头规范，是机制保障 | Lint→Git Hooks→CI/CD 全链路卡控 + Review 机制设计与推动 |

### Key Points

- **不要讲**：排期、分活、周报、1on1 这些"基层组长"的事
- **要讲**：技术决策的 why、推动落地的阻力与策略、量化结果
- **角度**：我是带着技术方向和产品视角做事的人，团队是我实现技术目标的方式，不是我的工作本身

### STAR Stories to Prepare

1. **研发资产沉淀**：搭建系统从 0→1，服务多个业务方向，物料调试效率提升 10x+
2. **质量治理推动**：从无规范到全链路卡控，如何说服团队、如何分步落地
3. **AI 体系规划**：为什么选 MCP 协议、如何串联全链路、落地效果
4. **跨团队协作**：与产品/后端/硬件的协作模式，如何对齐目标、解决分歧

## Impact Mindset

> 不出错是基础要求。Leader 的差异化在于**影响力半径**——你的产出能辐射多远。

### 三层影响力

| 层级 | 表现 | 对应动作 |
|------|------|---------|
| **团队内** | 团队效率提升、质量可控 | 基本盘，不值得单独讲 |
| **部门级** | 沉淀的基建被其他团队复用 | CLI/物料体系/搭建系统服务多个业务方向 |
| **公司级** | 产出工具/标准在公司普及，成为基础设施 | AI 工具链、MCP Server 暴露业务能力 |

### How to Tell the Story

**不要讲**："我管好了团队，按时交付了需求"（这是及格线）

**要讲**：
- 我沉淀的 X 被 Y 个团队复用，覆盖了 Z 个业务方向
- 我产出的 AI 工具从团队内部用 → 推广到部门 → 成为公司标准实践
- 我建设的 MCP Server 把业务能力暴露为标准协议，其他团队/AI Agent 可以直接调用，**让业务成为平台**

### Concrete Examples to Prepare

1. **基建普及**：搭建系统从服务自己团队 → 服务多个业务方向（物料 + CLI + CI/CD 全套）
2. **AI 工具产出**：MCP 全链路不只是自己用，是可以推广为团队/部门标准研发流程的
3. **业务 MCP 化**：把运营业务能力封装为 MCP Server，让 AI Agent 能直接操作业务（创建活动、配置规则、查询数据），**从"人操作系统"变成"AI 操作系统"**
4. **对外协作效率**：标准化接口 + 文档 + 工具链，让上下游对接成本从"开会对齐"变成"看文档自助"
