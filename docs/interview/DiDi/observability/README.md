## 核心 4 指标速记（跨端通用思路）

| 端 | 核心指标 | 一句话 |
|----|---------|--------|
| **Node.js** | P95 / 5xx / Event Loop Lag / 未捕获异常 | 快、稳、不卡、不崩 |
| **Web 前端** | LCP / CLS / INP / JS Error 率 | 加载快、不跳、交互流畅、不报错 |
| **搭建页面** | 渲染成功率 / 嵌套深度 / 组件数 / 白屏率 | 能渲出来、不失控 |
