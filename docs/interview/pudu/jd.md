# PD - Cross-platform App (Robotics)

## Responsibilities

1. 开发**跨平台 App**，提供与机器人交互的用户界面
2. 实现姿态控制与动作管理：固定姿态、远程触发预置脚本与动作序列，结合 TTS 增强交互体验
3. 引入**低延迟视频与控制方案**，确保遥控延迟低、稳定
4. 实现机器人状态仪表盘，实时展示传感器数据、关节姿态等信息
5. 与机器人控制系统进行高效通信，支持控制指令的发送与状态反馈
6. 撰写高质量前端代码，覆盖自动化测试、性能调优、持续集成
7. 与产品、硬件及后端团队协作，将功能需求转化为优质移动/桌面 App

## Requirements

1. 3年以上前端开发经验，计算机相关统招本科及以上
2. 精通 JavaScript/HTML/CSS 和 TypeScript
3. 熟练掌握跨平台框架（React Native / Flutter / Electron 等）
4. 了解实时通信协议（WebSocket / WebRTC / MQTT 等）
5. 有性能优化和自动化测试经验

## Prep Checklist

### Must-have
- [ ] 跨平台框架：RN / Electron 架构、通信机制、性能优化
- [ ] 实时通信：WebSocket 握手/心跳、WebRTC 信令流程、MQTT QoS
- [ ] 数据可视化：Canvas 渲染、大量实时数据的帧率优化
- [ ] TypeScript：类型体操、泛型、工具类型
- [ ] 工程化：CI/CD、自动化测试策略

### Bonus
- [ ] 视频流：低延迟方案（WebRTC SFU、WHIP/WHEP）
- [ ] 嵌入式/IoT 通信经验：串口、蓝牙、CAN 总线
- [ ] 快应用框架经验（JS 驱动 Native View，类似机器人控制场景）
- [ ] 状态管理：实时数据流 + UI 同步方案
