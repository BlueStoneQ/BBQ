# PD - Cross-platform App (Robotics)

## Responsibilities

1. 开发**跨平台 App**，提供与机器人交互的用户界面
2. 实现姿态控制与动作管理：固定姿态、远程触发预置脚本与动作序列，结合 TTS 增强交互体验
3. 引入**低延迟视频与控制方案**，确保遥控延迟低、稳定
4. 实现机器人状态仪表盘，实时展示传感器数据、关节姿态等信息
5. 与机器人控制系统进行高效通信，支持控制指令的发送与状态反馈
6. 撰写高质量前端代码，覆盖自动化测试、性能调优、持续集成
7. 与产品、硬件及后端团队协作，将功能需求转化为优质移动/桌面 App

## Requirements

1. 3年以上前端开发经验，计算机相关统招本科及以上
2. 精通 JavaScript/HTML/CSS 和 TypeScript；**精通 React Native**，能独立开发和发布应用
3. 熟练使用 Git 分支管理、CI/CD 流程，具有良好的 Git Commit 规范
4. 熟悉常用数据结构（栈/队列/链表/树/图/哈希）与经典算法，能在业务场景中灵活运用
5. 掌握 **Python 或 C++** 可优先考虑，具备服务端开发经验，并能独立排查常见问题
6. 编码风格严谨，追求可读性、可维护性与可测试性，能编写高质量单元测试用例

## Prep Checklist

### Must-have
- [ ] RN 精通：全链路开发 + 发布流程（Android/iOS 打包签名上架）
- [ ] 跨平台框架：RN + Electron 架构、通信机制、性能优化

- [ ] 数据结构与算法：链表/树/哈希/BFS/DFS（可能有手写题）
- [ ] TypeScript：类型体操、泛型、工具类型
- [ ] 单元测试：vitest/jest 用法、测试策略

### Bonus
- [ ] Python：自动化测试（pytest + uiautomator2）
- [ ] C++：JSI/JNI 架构理解、Pure C++ TurboModule
- [ ] 服务端：Node.js + 数据库 + API 设计
- [ ] 视频流：低延迟方案（WebRTC/WebSocket）
- [ ] IoT 通信：BLE/串口/MQTT
- [ ] 快应用框架经验（JS 驱动 Native View）
