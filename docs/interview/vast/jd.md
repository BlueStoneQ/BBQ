## 原文

1. 你将负责（Responsibilities）

- 主导核心功能的设计与交付 — 独立负责完整功能模块的技术方案、开发与上线，端到端 own 结果
- 构建高性能的 3D Web 体验 — 深入 Three.js 与自研 3D 引擎的集成，优化模型加载、渲染帧率与内存管理
- 守护和演进 SSR 架构 — 维护和优化 Nuxt 4 SSR 体系，保障服务端渲染稳定性、水合一致性和首屏性能
- 驱动 SEO 技术增长 — 负责结构化数据、SSR 友好的 URL 设计等技术 SEO 实践，用工程手段提升自然流量
- 设计可扩展的前端架构 — 主导状态管理、组合式函数体系、API 层抽象等关键架构决策，让代码库在快速迭代中保持健康
- 建设团队基础设施 — 推动自研组件库和工具链的演进，制定规范和最佳实践，提升团队效率
- 用 AI 重新定义开发效率 — 将 AI 编程工具深度融入工作流，探索 AI 辅助代码审查、测试编写等实践并推广到团队
- 保障工程质量的底线 — 完善 TypeScript 类型体系、测试覆盖和性能监控等质量保障机制
- 带动团队技术成长 — 通过 Code Review 传递经验，指导初级成员成长，参与技术分享和方案评审

2. 我们希望你具备（Requirements）

- 3-5 年前端开发实战经验 — 有从 0 到 1 或大型项目的完整经历，理解功能从需求到上线的全链路
- Vue 生态深度使用者 — 熟练掌握 Vue 3 Composition API、响应式原理、组件设计模式，有 Nuxt SSR 经验优先
- 理解 SSR 不只是加一层服务器 — 有服务端渲染实战经验，理解水合机制、跨请求状态隔离等核心概念，踩过坑并知道怎么避坑
- TypeScript 是你的默认选择 — 能编写类型安全的代码，善用泛型和类型推导，而不只是加个 `: string`
- 对性能有体感和手段 — 懂得分析和优化首屏加载、运行时性能和内存占用，有可量化的优化经历
- 工程化思维成熟 — 熟悉 Vite、CI/CD、代码规范和自动化测试，能在快速交付和代码质量之间找到平衡
- AI 是你的生产力倍增器 — 日常已习惯使用 AI 编程工具，能高效协作并对 AI 输出保持审慎判断
- 能扛事，也能讲清楚 — 独立推进复杂项目，遇到阻塞主动找方案；能清晰表达技术决策的取舍

- SSR 架构有深度实践 — 处理过水合不匹配、状态泄漏、SSR-safe 组件设计等真实问题

- 玩转 AI 辅助开发工作流 — 能用 AI 工具重构架构、生成测试、辅助 Code Review，并总结出可复制的方法论
- 有跨端或全栈经验 — 了解 Node.js / Nitro 运行时等服务端知识，或有桌面端开发经历
- 性能优化有代表作 — 能拿出具体案例和数据支撑
- 对 AIGC 和 3D 生成赛道有热情 — 关注过相关前沿方向，理解我们在做的事
