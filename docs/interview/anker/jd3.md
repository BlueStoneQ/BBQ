# Anker JD3 — 高级前端工程师（Web 图形编辑器 / 3D 渲染）PSP-前端开发类专项级五

> 深圳 · 已投递

## Responsibilities

1. 主导或深度参与图像编辑器、图文编辑工具、3D 渲染等工作
2. 承担 Web 前端核心模块的设计与开发，持续提升图像渲染效果，对代码质量及进度负责，持续提升编辑性能与稳定性
3. 与产品经理、设计师、后端研发紧密合作，高效高质地完成了页面的实现工作
4. 从技术视角推动用户体验和业务指标的持续优化，提供技术解决方案支持业务发展
5. 处理浏览器兼容问题，维护及优化前端页面性能，持续优化交互体验

## Requirements

1. 全日制统招本科以上，5 年以上前端开发经验，有大型复杂项目的前端架构设计经验者优先
2. 具备复杂前端架构设计经验，有状态机、插件化架构设计经验，能实现高内聚、低耦合的多功能前端框架，平衡性能、可维护性与用户体验
3. 精通 Web 前端技术：React、HTML5、JavaScript、TypeScript、CSS3、Sass、WebGL、Three.js
4. 有图形/图像相关编辑器经验，有 Fabric 画布编辑开发经验优先，有 3D 渲染项目经验者优先，能熟练使用 Canvas、Three.js/Babylon，掌握 WebGL、OpenCV 进行图像处理
5. 具备 AI 工具实际使用经验，能够熟练使用 Cursor、Claude 等 AI 编程辅助工具提升研发效率，有 AI 技术工程化应用经验者优先
6. 有强烈的好奇心、责任心和团队精神，善于沟通合作，学习能力强，能独立完成相关设计和编码

## Notes

- **强匹配点**：架构设计能力（状态机 + 插件化多项实践）+ AI 工具经验远超要求
- **3d-editor 项目**：已完成架构设计和技术选型（Fabric.js + Three.js + Zustand），但尚未完整实现。面试时可以讲设计思路和技术选型理由，但不能说"做过"
- **架构设计**：你有多个"状态机 + 插件化"的实践（Mako 微内核、探针 SDK 状态机、快应用 IDE 插件 API、CLI plugin 机制）
- **AI 工具**：不只是"会用"，而是"自己造了一个 AI Agent 框架"，远超 JD 要求
- **缺口**：
  - Fabric.js / Three.js 无实际产出 → 有架构设计和技术调研，但没有上线产品
  - OpenCV 图像处理无直接经验
  - WebGL 底层 shader 编程不深
- **风险**：如果面试深挖 Canvas/WebGL 实战细节（性能瓶颈怎么解决的、遇到过什么坑），可能答不上来
- **策略**：重点讲架构设计能力（可迁移）+ AI 工具（差异化），弱化"我做过编辑器"的表述，改为"我正在做 + 已完成架构设计"
