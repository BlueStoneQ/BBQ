# KS - Payment Frontend Engineer (E-commerce)

## Responsibilities

1. 负责支付业务的前端**技术选型和方案设计**、前端研发
2. 负责 H5/RN/Web 页面的**性能优化与体验提升**，利用可观测与数据化手段持续改进
3. 推动前端**工程化/组件化/自动化**建设，优化研发流程与工具链
4. 与后端/QA/产品/设计深度协作，主导技术方案设计与落地
5. 关注前沿技术与 **AI 技术发展趋势**，将 AI 能力融入开发流程与业务场景

## Requirements

1. 本科及以上，计算机相关，良好编程基础
2. 扎实 HTML/CSS/JS，熟悉 W3C 规范，熟练主流框架
3. 责任心 + 团队精神，能独立完成架构设计和编码落地
4. 熟悉至少一门后端语言（Node.js/Python）+ 项目实践（加分）
5. 中大型项目前端架构经验，支付/金融/信贷系统经验优先（加分）
