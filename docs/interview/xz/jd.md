## 原文

我们是一家科技公司(祥埻)，正在构建一款为家庭量身定做的金融成长系统 - Toggee.ai，其中融合了任务管理、数字资产、角色养成与沉浸式体验。

我们构建了双层产品架构：
- 现实层（React Native）负责家庭账户、业务流程及任务管理，提供简洁高效的操作体验；
- 梦想层（Unity）则通过角色成长与虚拟世界的互动，增强情感连接，提供情绪价值。

我们正在寻找一位优秀的App技术负责人，主导现实层（React Native）的整体技术架构与代码质量，解决各种复杂客户端问题。

### 核心职责

**1. React Native架构设计与代码质量**
- 评估并优化现有代码库，制定改进方案并主导实施。
- 设计清晰的模块边界（如任务、聊天、账户等），确保数据流可追踪。
- 选型并实施状态管理方案（如Zustand、Redux Toolkit），建立高效的开发标准和测试策略。
- 移动端性能优化：渲染性能、启动时间、内存管理等。

**2. 核心模块主导**
- 任务系统：设计任务列表、接取、提交和审批流程，确保信息密度高且操作路径清晰。
- 聊天系统：开发家庭聊天主通道，支持任务通知、数字资产变动推送等功能。
- 数字资产流程：实现余额展示、充值、历史记录和规则说明，确保系统的金融级可信赖感。
- 账户与设置：负责账户管理、家庭关系、订阅和权限控制等功能模块。

**3. 跨栈协作与边界管理**
- 理解并执行双层架构，确保React Native侧作为业务主场，Unity仅负责呈现层。
- 与Backend Lead密切协作，共同定义接口契约与状态同步机制。
- 管理React Native与Unity之间的转场体验，保证视觉语言的一致性和流畅度。

**4. 产品判断力与交互设计**
- 在没有完整设计稿的情况下，能独立做出合理的交互决策。
- 识别并避免可能损害用户信任感的设计方案，确保"工具感"与"效率感"的平衡。

**5. 团队管理与领导力**
- 带领App开发团队，进行代码审核与技术决策。
- 制定团队的工程质量标准，确保高质量的技术输出。
- 与Unity Lead、Back-end Lead协同工作，确保跨栈契约和技术标准的统一。

### 岗位要求

**1. 工作经验**
- 10年以上React Native / 移动端开发经验，具备主导产品上线的ownership经验。
- 有复杂状态系统（如任务流、聊天、账户、支付流程）的开发经验。
- 有接手并改善存量代码库的实际经验。

**2. 技术能力**
- 精通React Native及相关开发工具，具备优秀的代码结构设计能力。
- 具有状态管理（如Redux、Zustand等）经验，能够保证数据流可追踪。
- 熟悉移动端性能优化，能用数据说明优化效果。
- 理解后端边界，能清晰定义接口契约，避免将复杂业务逻辑放在客户端。

**3. 产品思维与UX判断力**
- 强烈的产品思维与用户体验判断力，能独立评估交互方案的合理性。
- 对"工具感"和"效率感"有清晰的产品直觉，能够为现实层提供简洁且高效的体验。

**4. 强加分项**
- Fintech经验：有支付、钱包、账户类App开发经验。
- 社交/通讯App经验：有聊天系统、消息推送、实时状态管理等相关经验。
- React Native与Unity混合架构经验：曾使用Unity作为React Native中的库。
- AI工具经验：熟悉AI coding tools（如GitHub Copilot）并已融入日常开发工作流。
- 小团队管理经验：有带领团队并制定工程标准的经验。
