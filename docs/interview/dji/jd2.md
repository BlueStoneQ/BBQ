
# DJI — 资深大前端架构师

> 深圳 · 软件 · 2026-05-31 发布

## Responsibilities

1. **大前端基础架构建设**：覆盖 Web、App、桌面端及跨端技术体系，统一框架、组件、工程化与研发体验
2. **公共能力建设**：Design System、脚手架、状态管理、埋点、国际化、包体/性能/崩溃治理、发布分发与观测
3. **多产品统一治理**：推动 Fly、Mimo、Studio、畅片、行业软件在工程规范、跨端复用、构建发布、质量门禁上统一
4. **跨端技术路线**：RN、Flutter/KMP、Electron、Web 等技术选型原则和落地策略
5. **端到云协同**：与后端基础架构、安全、效能/稳定性团队打通开发、测试、发布和运行时闭环
6. **团队建设**：大前端基础架构团队建设、关键技术攻坚、人才梯队培养

## Requirements

1. 本科+，8 年以上 Web/App/桌面端研发，3 年以上前端平台或客户端架构/团队管理
2. 大前端平台建设：组件库、脚手架、构建体系、发布体系、性能治理、质量平台（至少三个）
3. 熟悉 iOS/Android/Web/桌面端至少两个终端形态，能做跨端技术判断
4. 复杂应用的性能、稳定性、包体、崩溃、埋点、可观测性治理经验
5. 能推动多业务团队统一工程规范与基础能力，有影响力和落地能力
6. 加分：跨端技术栈、全球化应用、AI 辅助开发或端侧智能体验

## Notes

- **强匹配点**：性能治理（有数据）、质量平台、脚手架/工程化、AI 工具经验
- **核心卖点**：你做过的"条件编译 + 模块裁剪 + 包体优化"正好对应 DJI 多产品线的"跨端复用 + 包体治理"
- **缺口**：
  - 没有 Flutter/KMP 直接经验 → 但有 Electron + RN + 快应用 + Web 多端经验，跨端判断力够
  - Design System 没有从 0 建设过 → 可以讲组件分层 + 物料端的思路
- **定位**：这是一个平台架构岗，不是业务开发。重点讲"建设基础设施服务多个业务线"的经验
- **级别判断**：JD 要求偏高（P8/P9 级），强调影响力和多团队推动能力。你的团队规模（5 人）可能偏小，需要强调跨团队推动的经验（XT 质量治理推动全组采纳）

职位描述
工作职责：

1. 负责公司应用软件大前端/客户端基础架构建设，覆盖Web、App、桌面端及跨端技术体系，统一框架、组件、工程化与研发体验；

2. 建设公共UI/Design System、脚手架、状态管理、埋点、国际化、包体/性能/崩溃治理、发布分发与观测能力；

3. 推动Fly、Mimo、Studio、畅片、行业软件等产品在工程规范、跨端复用、构建发布、质量门禁上的统一治理；

4. 负责跨端技术路线评估与演进，形成原生 RN、Flutter/KMP、Electron、Web等技术选型原则和落地策略；

5. 与后端基础架构、安全、效能/稳定性团队协同，打通端到云的开发、测试、发布和运行时闭环；

6. 负责大前端基础架构团队建设、关键技术攻坚和人才梯队培养。



任职要求：

1. 本科及以上学历，8年以上Web/App/桌面端研发经验，3年以上前端平台或客户端架构/团队管理经验；

2. 具备大前端平台建设经验，做过组件库、脚手架、构建体系、发布体系、性能治理、质量平台等至少三个领域；

3. 熟悉iOS/Android/Web/桌面端至少两个终端形态，能做跨端技术判断而非单点优化；

4. 有复杂应用的性能、稳定性、包体、崩溃、埋点、可观测性治理经验；

5. 能推动多业务团队统一工程规范与基础能力，具备较强的影响力和落地能力；

6. 有跨端技术栈、全球化应用、AI辅助开发或端侧智能体验建设经验者优先。
