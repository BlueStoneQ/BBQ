# RN插件开发
- 安装脚手架
```bash
npm i -g react-native-create-library
```
- 脚手架创建rn插件项目
```bash
# 教程 https://www.jianshu.com/p/d9ab220d9288
react-native-create-library --platform ios,android quicklogin-plugin-rn-youmeng
```
